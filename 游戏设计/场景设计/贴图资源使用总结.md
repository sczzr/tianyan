## 🎮 使用方法

### 在Godot中打开

1. 打开Godot编辑器
2. 加载项目
3. 在文件系统中找到 `Scenes/Locations/`
4. 双击 `TianYanMainPeak_AssetBased.tscn`

### 运行场景

```
按 F6 运行当前场景
```

### 控制方式

```
WASD / 方向键 - 移动
E - 交互
鼠标滚轮 - 缩放
鼠标中键 - 拖拽相机
鼠标移到边缘 - 边缘滚动
```
