# 天玑府

## 府级画像

【广袤的战争熔炉】 拥有全图第二大的广袤面积，宗门林立，内部摩擦不断。这种环境极其锻炼修士的实战能力。天玑府是北朔州联盟最主要的“兵源地”，其出产的修士虽然顶尖高手比例稍低，但个个都是身经百战的精锐。
