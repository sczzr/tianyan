# 天权府

## 府级画像

【保守的元老院】 修仙传承可能更为古老和保守，招收弟子的标准极为严苛，不盲目扩张。这里的宗门可能是北朔州联盟的“元老院”，由最古老的几个修仙世家或宗门构成。
