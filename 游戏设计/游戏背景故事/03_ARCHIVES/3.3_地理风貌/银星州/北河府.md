# 北河府

## 府级画像

【昔日帝都，遗落之城】 北河府曾是银星州辉煌时代的都城，如今只剩下巨大的废墟和断壁残垣。城中流传着无数关于上古宝藏和失落传承的传说，吸引着大量外来的寻宝者和散修。这里的本土势力，则在守护着最后的荣耀与秘密。
