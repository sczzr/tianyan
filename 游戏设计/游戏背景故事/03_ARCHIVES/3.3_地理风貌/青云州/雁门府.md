# 雁门府

## 府级画像

【实力雄厚的北方双雄】 与江陵府同为青云州北部的两大霸主，常年对峙。雁门府可能更靠近内陆，作风勇猛。

## 核心矛盾

两强争霸，谁能统一青云州北部。
