# 南泽州 (Southmarsh State)

## 州级画像

世界的霸主 (The Hegemon)：苍玄界的绝对核心。拥有最多的顶级宗门和最庞大的人口，是政治、经济和修仙文明的巅峰。资源高度集中，导致了地狱级的内部竞争。
