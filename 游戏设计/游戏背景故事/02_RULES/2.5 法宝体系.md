### 法宝体系

#### 1.1 法宝基础属性

| 法宝类型   | 定义与形成                                                                 | 驱使方式         | 认主情况                     | 是否可放入体内 | 核心特点                                                                 | 是否可被他人抹去印记 | 是否可祭炼为本命法宝 | 受损是否反伤宿主 |
|------------|----------------------------------------------------------------------------|------------------|------------------------------|----------------|--------------------------------------------------------------------------|----------------------|----------------------|------------------|
| **法宝**   | 炼器师用材料炼制，烙下神识印记即可驱使                                     | 神识或灵力       | 可被他人抹去印记（可更换认主） | 是             | 基础法宝，无特殊成长性                                                   | 是                   | 是（通过神魂祭炼）   | 否               |
| **本命法宝** | 普通法宝经神魂祭炼后形成                                                   | 神识或灵力       | 认主后不可更换               | 是             | 威力随宿主修同步提升，与宿主生命绑定                                     | 否                   | 它本身就是本命法宝   | 是               |
| **先天灵宝** | 法宝铸成时自主生成器灵                                                     | 神识或灵力       | 可主动认主                   | 是             | **功能性极强**（如辅助修炼、空间穿梭等），器灵独立存在                   | 是                   | 是（但会抹去器灵）   | 否               |
| **后天灵宝** | 本命法宝主人死后，法宝催生器灵形成                                         | **仅灵力**       | 不可认主（无主状态）         | **否**         | **杀伤性极强**（如毁灭性攻击），器灵残留战斗本能                         | 否（无主）           | 否                   | 否               |
| **古宝**     | 本命法宝主人死后，法宝未产生器灵                                           | **仅灵力**       | 不可认主（无主状态）         | **否**         | **杀伤性极强**（如规则攻击），但无器灵加持，状态更不稳定                 | 否（无主）           | 否                   | 否               |

#### 1.2 法宝境界适用与威力差异

| 法宝类型   | 练气期 | 筑基期 | 金丹期 | 元婴期 | 化神期 | 威力描述（按境界） |
|------------|--------|--------|--------|--------|--------|------------------|
| **法宝**   | 下品法器<br>（基础攻击/防御） | 中品法器<br>（增强攻击/防御） | 上品法器<br>（大范围攻击/防御） | 法宝<br>（元素操控） | 高阶法宝<br>（法则初显） | 练气：相当于练气修士全力一击<br>筑基：相当于筑基修士全力一击<br>金丹：相当于金丹修士全力一击<br>元婴：相当于元婴修士全力一击<br>化神：相当于化神修士全力一击 |
| **本命法宝** | - | - | 初级本命<br>（与金丹修士绑定） | 成熟本命<br>（与元婴修士绑定） | 完美本命<br>（与化神修士绑定） | 金丹：相当于金丹修士全力一击<br>元婴：相当于元婴修士全力一击<br>化神：相当于化神修士全力一击<br>**特点**：随主人境界提升而成长，可进化 |
| **先天灵宝** | - | - | - | 初级灵宝<br>（发挥部分功能） | 完整灵宝<br>（发挥全部功能） | 元婴：相当于元婴修士全力一击<br>化神：相当于化神修士全力一击<br>**特点**：功能性极强，如空间穿梭、时间加速、法则辅助等 |
| **后天灵宝** | - | - | - | 初级灵宝<br>（器灵本能） | 完整灵宝<br>（器灵觉醒） | 元婴：相当于元婴修士全力一击<br>化神：相当于化神修士全力一击<br>**特点**：杀伤性极强，器灵残留战斗本能，无智能 |
| **古宝**     | - | - | - | 初级古宝<br>（规则初显） | 完整古宝<br>（规则掌控） | 元婴：相当于元婴修士全力一击<br>化神：相当于化神修士全力一击<br>**特点**：杀伤性极强，蕴含规则之力，但状态不稳定 |

#### 1.4 法宝功能差异（按境界）

| 法宝类型   | 练气期 | 筑基期 | 金丹期 | 元婴期 | 化神期 | 功能特点 |
|------------|--------|--------|--------|--------|--------|---------|
| **法宝**   | 单一元素攻击<br>基础防御 | 多元素组合<br>增强防御 | 大范围攻击<br>高级防御 | 元素操控<br>空间稳定 | 法则初显<br>规则应用 | 低阶功能单一，高阶功能多样，化神期可初步应用法则 |
| **本命法宝** | - | - | 与主人绑定<br>基础成长 | 与主人共鸣<br>功能进化 | 与主人合一<br>法则共鸣 | 随主人境界提升而进化，功能从单一到复合，最终可触及法则 |
| **先天灵宝** | - | - | - | 辅助修炼<br>空间穿梭<br>时间加速 | 法则辅助<br>小世界开辟<br>跨界传送 | 功能性极强，化神期可开辟小世界，跨界传送 |
| **后天灵宝** | - | - | - | 毁灭性攻击<br>能量吞噬<br>领域压制 | 规则攻击<br>能量湮灭<br>法则破坏 | 杀伤性极强，化神期可施展规则级攻击 |
| **古宝**     | - | - | - | 规则攻击<br>空间切割<br>时间扭曲 | 规则掌控<br>时空操控<br>因果干涉 | 杀伤性极强，化神期可操控时空规则，干涉因果 |
