# 2.14 战斗力修正体系 (V4)

本文档定义了在“战斗力基准”（由境界和等级决定）之上，用于计算角色综合战斗力的各项修正系数。这些系数旨在量化功法、法宝、战技等外部因素对角色实战能力的影响。

## 核心公式

**综合战斗力 = 战斗力基准 × 功法系数 × 法宝系数 × 战技系数 × 神念系数 × (1 + 其他特殊修正)**

*注：符箓通常作为一次性爆发手段，其效果可视为临时提高最终的“综合战斗力”，而非永久修正系数。*

---

## 1. 功法修正系数

根据角色主修功法的品阶来决定。

| 功法品阶 | 修正系数范围 | 备注 |
| :--- | :--- | :--- |
| 黄阶 | 1.05 - 1.15 | 大路货，基础中的基础 |
| 玄阶 | 1.20 - 1.30 | 宗门核心弟子标配 |
| 地阶 | 1.35 - 1.50 | 镇派之宝，非天才或长老不得传授 |
| 天阶 | 1.60 - 1.80 | 传说中的功法，通常有残缺或苛刻的修炼条件 |

## 2. 法宝修正系数 (参照 2.5 法宝体系.md)

根据角色最核心的战斗法宝决定。**所有法宝的使用均无境界限制，但越级使用会大幅增加灵力消耗，且无法发挥全部威力。**

#### 2.1 标准法宝

| 法宝层级 | 常规使用境界 | 修正系数范围 | 越级使用代价 |
| :--- | :--- | :--- | :--- |
| 下品法器 | 练气期 | 1.05 - 1.10 | 灵力消耗增加 |
| 中品法器 | 筑基期 | 1.11 - 1.15 | 灵力消耗巨大 |
| 上品法器 | 金丹期 | 1.16 - 1.25 | 灵力消耗巨大，可能导致虚脱 |
| 普通法宝 | 元婴期 | 1.30 - 1.40 | 灵力消耗极巨，有反噬风险 |
| 高阶法宝 | 化神期 | 1.45 - 1.60 | 几乎无法驱动，强行使用会受重伤 |

#### 2.2 特殊法宝

| 法宝类型 | 修正系数 | 备注 |
| :--- | :--- | :--- |
| **本命法宝** | [同阶标准法宝系数] × 1.1 | 与宿主神魂相连，威力额外提升10%且随宿主成长。 |
| **灵宝/古宝** | 1.70+ | **【元婴期及以上方可使用】** 剧情道具，威力逆天，其系数应单独评估。 |

## 3. 符箓修正 (临时爆发，参照 2.6 符箓体系.md)

符箓的使用没有境界限制，但使用者修为越低，催动高阶符箓的代价越大。

#### 3.1 普通符箓 (一次性)
| 符箓品阶 | 常规使用境界 | 威力加成 (临时) |
| :--- | :--- | :--- |
| 低级符箓 | 练气期 | 临时提升5%-10%战斗力 |
| 中级符箓 | 筑基期 | 临时提升10%-15%战斗力 |
| 高级符箓 | 金丹期 | 临时提升15%-25%战斗力 |
| 顶级符箓 | 元婴期 | 临时提升25%-40%战斗力 |
| 神级符箓 | 化神期 | 临时提升40%-60%战斗力 |

#### 3.2 符宝 (可重复，威力为原法宝的1/10)
*   **定位**：相当于一个弱化版的、可多次使用的“大招”。
*   **修正方式**：使用时，可进行一次威力相当于 **[制作者境界]** 的攻击，其战斗力数值为 **[制作者战斗力基准] × 0.1**。越级使用消耗巨大。

#### 3.3 真宝 (一次性，威力为原法宝的1/3)
*   **定位**：威力巨大的杀手锏，但有严苛的使用限制（通常为直系血脉）。
*   **修正方式**：使用时，可进行一次威力相当于 **[制作者境界]** 的强力攻击，其战斗力数值为 **[制作者战斗力基准] × 0.33**。

| 融会贯通 | 1.16 - 1.25 |

## 5. 神念修正系数

神念强度决定了修士的感知范围、控制精度和对幻术/精神攻击的抵抗能力。一个强大的神念可以在战斗中占据巨大优势。

| 神念强度 (相对自身境界) | 修正系数范围 | 描述 |
| :--- | :--- | :--- |
| 神魂受损 | 0.85 - 0.95 | 反应迟钝，控制法宝易出错，极易陷入幻境。 |
| 弱于同阶 | 0.95 - 1.00 | 感知范围较小，控制多件法宝或复杂法术时略显吃力。 |
| **同阶平均** | **1.00** | 正常水平，无加成也无短板。 |
| 强于同阶 | 1.05 - 1.15 | 感知敏锐，能提前预判危险，控制法宝更加精准。 |
| 远超同阶 | 1.20 - 1.30 | 可多线操作法宝与法术，甚至能以神念直接压制低阶修士。 |
| 神念化形 | 1.35+ | **【剧情设定】** 神念凝如实质，可发动神识攻击，极其罕见。 |

## 6. 其他特殊修正
*   **丹药加成**: +0.2至+0.5
*   **阵法加成**: +0.1至+0.8
*   **血脉/体质**: +0.1至+1.0
*   **克制关系**: -0.1至-0.5
