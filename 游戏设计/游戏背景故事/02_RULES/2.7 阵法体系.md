### **阵法体系**

#### **1. 阵法定义与分类**

阵法是修士利用特定材料，在特定位置布置，引动天地灵气，以达到特定目的的法则具象化手段。

| 阵法类型 | 核心功能 | 主要应用场景 | 代表阵法 |
| :--- | :--- | :--- | :--- |
| **守护阵** | 防御、结界、隐匿 | 宗门护山、洞府守护、重要区域保护 | 护山大阵、九宫八卦阵 |
| **聚灵阵** | 汇聚、提纯灵气 | 修炼室、灵田、药园、炼丹/炼器室 | 聚灵阵、引脉阵 |
| **迷踪阵** | 幻术、困敌、遮蔽 | 秘境入口、防止外人闯入、战场辅助 | 迷踪阵、幻海阵 |
| **杀伐阵** | 攻击、绞杀、削弱 | 战场、陷阱、宗门防御反击 | 剑气杀阵、天雷劫罚阵 |
| **辅助阵** | 传送、通讯、辅助生产 | 远距离传送、宗门通讯、辅助炼丹炼器 | 传送阵、通讯阵、恒温阵 |

#### **2. 布阵核心要素**

一个完整的阵法通常由以下几个部分构成，其品质决定了阵法的最终威力与稳定性。

| 核心要素 | 定义与作用 | 常用材料（示例） |
| :--- | :--- | :--- |
| **阵眼** | 阵法的核心，控制整个阵法的运转与能量输出 | 灵矿核心、高阶妖丹、顶级法宝、修士自身 |
| **阵基** | 承载阵法力量的基石，决定了阵法的稳定性和持久性 | 巨型灵矿核心、九天息壤、大地龙脉 |
| **阵旗/阵盘** | 引导灵力流向、切换阵法变化的媒介 | 高级灵木板、玄冰玉髓、空青木 |
| **能源核心** | 为阵法提供运转所需能量的源泉 | 灵石矿脉、极品灵石、高阶妖丹 |

#### **3. 阵法师体系**

阵法师是修仙百艺中对天赋和计算能力要求极高的职业，地位尊崇。

- **入门阵法师**：能布置**练气期**、**筑基期**级别的基础阵法（如聚灵阵、小型迷踪阵），通常服务于小型家族或商铺。
- **高阶阵法师**：能布置**金丹期**、**元婴期**级别的复杂大阵（如护山大阵、杀伐阵），是各大宗门争抢的核心人才。
- **阵道宗师**：能布置**化神期**及以上，甚至跨州传送的战略级阵法，其存在本身就是一个宗门的底蕴。

#### **4. 价值体系**

- **设计成本**：高级阵法图纸本身就是无价之宝，通常为宗门不传之秘。
- **材料成本**：根据阵法规模和等级，材料耗费从数百上品灵石到数条灵脉不等，是修仙界最昂贵的工程之一。
- **维护成本**：大型阵法需要定期补充能源（灵石），并由阵法师进行维护，是一笔持续的巨大开销。

#### **5. 聚灵—提纯—自适应（机制说明）**

- 为什么需要“提纯”：天地灵气是混合物，多灵根修士吸入后需在体内完成“分离/提纯/排出/转化”的耗时流程。聚灵阵在体外做“预处理”，可显著降低内炼成本。
- 标准聚灵阵：以“聚”为主，汇拢并初步提纯灵气，适用于绝大多数修炼空间（修炼室/药园/炼丹室）。
- 自适应聚灵阵：在“聚”的基础上，引入“节拍锁相/短促聚焦/心域静稳”等控制手段，将阵图参数与个体体征/环境状态相适配，进一步提升单位时间有效吸收，降低“回流/刺返”。
