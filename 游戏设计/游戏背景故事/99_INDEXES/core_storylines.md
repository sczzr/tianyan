# 主线集合（Core Storylines）

说明
- 此文档绑定全书主线至顶层 QCHZ 子树，作为卷纲/Unit/Beats/章纲的上游约束。
- 默认分类：明线 / 暗线 / 成长 / 情感；可扩展。

---

## 明线 SL-MAIN
- 绑定：BOOK 全段（Q→C→Z→H）
- 最高目标：以工程化方法重构修行体系，并通过制度化/标准化落地
- 卷级回收（示例）：
  - Vol.1–2 = 播（方法雏形/证据初始化）
  - Vol.3–6 = 土（品控/流程/讲学/制度触点）
  - Vol.7–12 = 土→果（标准/网络/军备合作雏形）
  - Vol.13–18 = 果（联盟与蓝图承载）
  - Vol.19–22 = 总果（补天执行层）
- 关键证据：讲法堂实验、抽检数据、SOP、阵网论文/报告

## 暗线 SL-DARK
- 绑定：BOOK-C / BOOK-Z / BOOK-H
- 最高目标：追溯“天道防火墙”的来源、结构与漏洞，并获取副本
- 卷级回收（示例）：
  - Vol.4–6 = 播（档案端倪/遗迹线索）
  - Vol.7–10 = 土（情报攻防/渗透）
  - Vol.11–12 = 果（副本在手）
  - Vol.13–18 = 土（档案解读/临停谈判）
  - Vol.19–22 = 果（最终版升级与执行）
- 关键证据：档案片段、遗迹样本、谈判纪要、系统日志

## 成长 SL-GROWTH
- 绑定：BOOK 全段
- 最高目标：从研究者→技术领袖→联盟总设计师→启明道祖
- 回收准则：每卷至少一项“技能/制度/资源/修为”达成果实态（记账式盘点）
- 关键证据：修为台账、技能清单、组织制度文本、里程碑清单

## 情感 SL-RELATION
- 绑定：BOOK-C / BOOK-Z / BOOK-H
- 最高目标：以“智力互信”为纽带建立同盟与伙伴（如苏沐/柳青青）
- 卷级回收（示例）：C 段播 → Z 段土 → H 段果（同盟/共同蓝图/契约）
- 关键证据：共同论文、工坊契约、协作蓝图、关键战役并肩记录
