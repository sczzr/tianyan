 ## 明线 SL-MAIN

 - 最高目标：建立“回档经济学/时间方法学”，用有限的三次回档获得文明最优解
 - 回收窗口（示例）：
   - Q 段 = 播（账本雏形/首审过线）
   - C 段 = 土（模型定型/联盟与标准）
   - Z 段 = 土→果（蓝图形成与对赌胜出）
   - H 段 = 总果（文明回档执行与可继承账本）
 - 证据：论文、数据、审计报告、伦理准则、公开辩论记录
