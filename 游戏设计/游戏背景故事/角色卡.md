### **世界书模板：全角色名册**

**【Codex AI 操作手册 - 用于生成 `lorebook_characters.json`】**

#### **文件名**
`lorebook_characters.json` (或任何您喜欢的描述性名称，如 `世界主要人物名册.json`)

#### **文件根结构**
```json
{
  "spec": "lorebook_v3",
  "data": {
    // ... 核心字段在此填充 ...
  }
}
```

#### **核心字段 (`data` 对象) 规则**

*   **`name`:** "【您的世界名】 - 主要人物名册"
    *   *示例: "维斯特洛大陆 - 主要人物名册"*
*   **`description`:** "本文件收录了故事中所有重要角色的基本信息、身份背景以及与他人的核心关系。"
*   **`scan_depth`:** 10
*   **`token_budget`:** 4096 (可以设置得高一些，因为可能同时触发多个角色)
*   **`recursive_scanning`:** `false`
*   **`entries`:** [ ... 见下文 ... ]

---

#### **角色条目 (`entry`) 生成规则与范例**

世界书的 `entries` 数组中，**每一个对象都代表一个独立的角色**。Codex AI应为每个角色笔记生成一个遵循以下规则的`entry`。

##### **`entry` 字段规则：**

*   **`id`:** `char_[角色名的小写拼音或英文名]`
    *   *示例: `char_alwin`, `char_lena`*
*   **`comment`:** "关于角色【角色名】的基本信息档案。"
    *   *示例: "关于角色莉娜的基本信息档案。"*
*   **`keys`:**
    *   **必须包含**：角色全名、昵称、姓氏、名字、身份称谓（如“队长”、“公主”）、外号、以及可能的代称（如“那个黑发女孩”）。关键词越全，触发越准。
*   **`content`:**
    *   这是**人物小传**的核心。内容应以**客观、简洁的第三人称**编写，包含以下**标准模块**：
        1.  **【姓名/身份】:** 全名，以及TA最广为人知的身份。
        2.  **【外貌】:** 2-3个最显著的外貌特征。
        3.  **【性格】:** 3-5个核心性格关键词。
        4.  **【背景/立场】:** 简述其出身、所属组织、当前立场或目标。
        5.  **【与[主视角角色]的关系】:** **(关键部分)** 清晰地描述这个角色与故事主视角人物（例如艾尔文）的关系、历史和当前状态。
*   **`insertion_order`:** 建议为所有角色条目设置一个相近的`insertion_order`值（例如 200-300之间），以确保它们作为同类信息被处理。

---

#### **范例：艾尔文世界中的“全角色名册”**

假设您的主视角角色是艾尔文，您的Codex AI读取了莉娜、罗德里克、以及一位新角色——精灵长老“埃拉里昂”的笔记后，应生成如下`lorebook_characters.json`：

```json
  "data": {
    "name": "银月森林 - 主要人物名册",

    "entries": [
      {
        "id": "char_lena",
        "comment": "关于角色莉娜的基本信息档案，艾尔文的妹妹。",
        "keys": [
          "莉娜", "妹妹", "艾尔文的妹妹", "Lena", "那个黑发女孩"
        ],
        "content": "【姓名/身份】: 莉娜 (Lena)，艾尔文失散多年的妹妹。\n【外貌】: 黑发及腰，右眼下方有一颗泪痣，身形瘦弱。\n【性格】: 活泼、善良、充满好奇心、坚韧。\n【背景/立场】: 在家乡被毁时失踪，是艾尔文踏上旅途的根本原因。目前生死不明。\n【与艾尔文的关系】: 艾尔文唯一的亲人，是他内心最柔软、最深沉的牵挂和愧疚所在。艾尔文坚信她还活着，并以找到她为最高目标。",
        "enabled": true,
        "insertion_order": 200
      },
      {
        "id": "char_roderick",
        "comment": "关于角色罗德里克的档案，艾尔文的前队长与仇人。",
        "keys": [
          "罗德里克", "队长", "Roderick", "那个背叛者", "铁狼"
        ],
        "content": "【姓名/身份】: 罗德里克，‘铁狼’佣兵团的前任团长，黄金级佣兵。\n【外貌】: 身材高大魁梧，左脸有一道从额头到下巴的刀疤，眼神锐利而冷酷。\n【性格】: 极具领袖魅力、唯利是图、冷酷无情、善于权衡利弊。\n【背景/立场】: 为了自保而背叛了艾尔文的部队，现已不知所踪，可能在为某个大势力效力。\n【与艾尔文的关系】: 曾经的上下级和战友，现在是艾尔文不共戴天的仇人。艾尔文对他怀有极深的憎恨和不信任。",

        "insertion_order": 201
      },
      {
        "id": "char_elarion",
        "comment": "关于精灵长老埃拉里昂的档案，潜在的盟友或信息提供者。",
        "keys": [
          "埃拉里昂", "精灵长老", "银月长者", "Elarion"
        ],

        "enabled": true,
        "insertion_order": 202
      }
    ]
  }
}
```
